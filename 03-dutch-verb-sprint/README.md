# Dutch Verb Sprint

Dutch Verb Sprint is a small desktop flashcard game for practising common Dutch verbs.

The app shows an English infinitive such as `to eat`. Type the Dutch infinitive, such as `eten`, then press Enter or click **Check**. Correct answers increase your score and streak. Wrong answers show the correct answer and return to the question queue so they appear again soon.

## Features

- English-to-Dutch verb practice
- Around 500 useful everyday Dutch verbs
- Instant feedback after each answer
- Score, attempts, accuracy, and streak
- Skip, reveal answer, next, and reset controls
- Last 5 answers summary
- Uses only Python and `tkinter` from the standard library

## How to run

Open a terminal in this folder and run:

```bash
python main.py
```

No external packages are needed.
