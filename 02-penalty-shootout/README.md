# World Cup Penalty Shootout

A simple desktop penalty shootout game built with Python and Tkinter.

## Requirements

- Python 3
- No extra packages are needed. The game only uses the Python standard library.

## How to Run

Open a terminal in this folder and run:

```bash
python main.py
```

On some systems, you may need to use:

```bash
python3 main.py
```

## How to Play

Choose `Left`, `Centre`, or `Right` for each penalty.

The goalkeeper randomly dives left, centre, or right. If the goalkeeper dives the same way as the shot, the penalty is saved. Otherwise, it is a goal.

Each side takes five penalties. If the score is tied after five each, the game continues into sudden death.

Use `New Game` to restart the shootout.
